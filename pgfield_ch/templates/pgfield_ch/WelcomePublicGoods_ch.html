这是本次活动的第二个游戏。

在这个部分，您会与其他{{ n_minus_one}}个人随机组成一个小组，编号1、2、3、4。
开始时，你们每个人将得到{{ Constants.endowment }}初始筹码。

第一环节: 公共账户投资

您需要决定要从您的{{ Constants.endowment }}筹码中拿出多少给公共账户。
这完全取决于您。
剩下的筹码将留在您的私人账户上。
您和组里其他人给公共账户的所有筹码，我们都将乘以{{ Constants.effic_factor }}后放入公共账户。
不论你们分别往公共账户里放了多少筹码，
公共账户的所有筹码将在组里{{ Constants.players_per_group }}个人中平分。
最终您能从公共账户里拿到的筹码数由您和组里其他三个人的选择决定。

例如，让我们假设您给了公共账户 {{ endowment_2|floatformat:0 }}点筹码， 而组里其他 {{ n_minus_one }} 个人总共给了公共账户{{ contrib_others_high|floatformat:0 }}点筹码， 您将从公共账户中分到 (21+5) x {{ Constants.effic_factor }} / {{ Constants.players_per_group }} = {{ payoff_ex_high|floatformat:1 }} 点筹码。 这些筹码将和您私人账户中的{{ endowment_2|floatformat:0 }}点筹码加总，因此您的总收益是{{ payoff_high|floatformat:0 }}点筹码。

现在仍然假设您给了公共账户{{ endowment_2|floatformat:0 }}点筹码， 但是组里其他 {{ n_minus_one }} 个人总共给了公共账户{{ contrib_others_low|floatformat:0 }}点筹码， 您将从公共账户中分到(9+5) x {{ Constants.effic_factor }} / {{ Constants.players_per_group }} = {{ payoff_ex_low|floatformat:1 }} 点筹码。 这些筹码将和您私人账户中的{{ endowment_2|floatformat:0 }}点筹码加总，所以您的总收益是{{ payoff_low|floatformat:0 }}点筹码。

第二环节：发反对卡

在这一环节开始时， 您会看到组里其他人在上一环节分别给公共账户多少筹码。 您需要决定要不要给组里其他人发一张反对卡。 您每发一张反对卡给别人，您的收益将减少 {{ Constants.punish_cost }} 点筹码。
收到反对卡的人的收益将减少 {{ Constants.punish_tech }} 点筹码。
只要他们想，组里其他人也可以发反对卡给您。

针对组里每个人，您都需要选择“发”或“不发”一张反对卡给他/她，。
您发的反对卡越多，您出的成本就越高。


例如， 假设您总共发了两张反对卡
而您被发了一张反对卡，
您的总收益将因此减少9点：为了给组里两个人发反对卡，您需要付4点筹码；同时，您因为被发一张反对卡而失去5点筹码。
