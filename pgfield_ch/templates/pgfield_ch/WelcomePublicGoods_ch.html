这是本次活动的第二部分。

在第二部分包含五轮重复实验。
在实验开始前，电脑会随机将您与其他{{ n_minus_one}}名参与者组成一个小组。
需要注意的是，小组成员在五轮实验中保持不变。
每轮实验开始时每个人将得到{{ Constants.endowment }}初始筹码。
所有五轮实验的结果都会被计入最终收益。

第一环节: 公共账户投资

您需要把{{ Constants.endowment }}筹码在公共账户和您的私人账户之间分配。 分配多少筹码给公共账户完全取决于您。
剩下的筹码将留在您的私人账户上。
您和组内其他参与者分配给公共账户的所有筹码我们都将乘以{{ Constants.effic_factor }}，作为公共账户的投资回报。
所有公共账户的投资回报将在{{ Constants.players_per_group }}个人中平分。

例如，如果您分配了 {{ endowment_2|floatformat:0 }}点筹码给公共账户， 而组内其他 {{ n_minus_one }} 个参与者总共分配了{{ contrib_others_high|floatformat:0 }}点筹码给公共账户， 您将从公共账户中得到(21+5) x {{ Constants.effic_factor }} / {{ Constants.players_per_group }} = {{ payoff_ex_high|floatformat:1 }} 点筹码。 这些筹码将和您私人账户中的{{ endowment_2|floatformat:0 }}点筹码加总，作为您的总收益。 所以您的总收益是{{ payoff_high|floatformat:0 }}点筹码。

如果您还是分配了 {{ endowment_2|floatformat:0 }}点筹码给公共账户， 而组内其他 {{ n_minus_one }} 个参与者总共分配了{{ contrib_others_low|floatformat:0 }}点筹码给公共账户， 您将从公共账户中得到(9+5) x {{ Constants.effic_factor }} / {{ Constants.players_per_group }} = {{ payoff_ex_low|floatformat:1 }} 点筹码。 这些筹码将和您私人账户中的{{ endowment_2|floatformat:0 }}点筹码加总，作为您的总收益。 所以您的总收益是{{ payoff_low|floatformat:0 }}点筹码。

第二环节：分配反对筹码

在这一环节， 您将看到组内其他参与者在前一环节中分配了多少筹码给公共账户。 您需要决定给每个组内成员分别分配多少反对筹码。 您每分配1点反对筹码给组内一个参与者，您将损失1点筹码，而收到反对筹码的参与者将损失3点筹码。 同样的，组内其他人也可以把反对筹码分配给您。

您分配给其他三个人的反对筹码的总和不能超过{{ Constants.endowment }}。
如果您不想给某个组内成员分配反对筹码，请输入 0。

例如， 如果您决定分别给组内其他三个参与者分配 0、1、1 点反对筹码，
而您总共收到了来自组内其他成员的1点反对筹码，
您的总收益将减少5点：您支付了2点筹码来投2点反对筹码给其他成员，另外，您因为收到1点反对筹码而损失3点筹码。

请耐心等待，等听到工作人员的指示后，再点击“下一步”按钮。
